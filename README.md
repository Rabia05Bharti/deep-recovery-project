# Deep Recovery Project

This project demonstrates basic disk recovery techniques using Linux tools.

## Objective
Recover files from disk images formatted with different file systems.

## Disk Images
Two disk images were created locally:
- disk_btrfs.img
- disk_xfs.img

These images are **not uploaded to GitHub** because they exceed the 100MB file limit.

## Tools Used
- dd
- mkfs.btrfs
- mkfs.xfs
- mount
- file recovery tools

## Steps Performed
1. Created disk images
2. Formatted them with BTRFS and XFS
3. Mounted the images
4. Created and deleted test files
5. Attempted file recovery
